# 最终验收清单

## A. 输入完整性

- [ ] PRD / IA 已确认
- [ ] 文案已定稿
- [ ] 定价已确认
- [ ] 合规边界已确认
- [ ] 至少看过 2-3 个竞品

## B. 页面覆盖

- [ ] 首页 Desktop
- [ ] 首页 Mobile
- [ ] Pricing / Features / FAQ 等关键页
- [ ] SEO 矩阵页模板
- [ ] 404 / Legal 等基础页按需覆盖

## C. 反 AI 味

- [ ] 字体不是 Inter / Roboto / Arial
- [ ] 不是紫蓝渐变 + 白背景
- [ ] 不是居中 Hero + 3 列 Features
- [ ] 没有 emoji icon
- [ ] 没有抽象 3D 球体占位
- [ ] 没有 Revolutionize / Empower / Seamless 等空话
- [ ] 有明确调性，不是“通用科技感”

## D. 转化检查

- [ ] 首屏 5 秒内能看懂产品
- [ ] 首屏 CTA 可见
- [ ] 页面中部有第二个 CTA
- [ ] 页面底部有最终 CTA
- [ ] Pricing 高亮方案清楚
- [ ] FAQ 回答关键疑虑

## E. 视觉基础

- [ ] 信息层级清楚
- [ ] Section 间距足够
- [ ] 正文对比度 ≥ 4.5:1
- [ ] 大标题对比度 ≥ 3:1
- [ ] 同类组件规则一致
- [ ] 配色不超过 3 个主色

## F. 移动端

- [ ] 375px 无横向滚动
- [ ] 标题不超过 2 行或仍可读
- [ ] 按钮点击区域 ≥ 44×44px
- [ ] 3 列布局变 1 列
- [ ] 图片不撑爆屏幕
- [ ] 移动端 CTA 可见

## G. 开发交付

- [ ] `HANDOFF.md` 已生成
- [ ] Design Tokens 完整
- [ ] Logo / Favicon / OG Image 已有或有提示词
- [ ] 按钮链接不再是 `#`
- [ ] 临时图片已标记或替换
- [ ] 开发注意事项写清楚
