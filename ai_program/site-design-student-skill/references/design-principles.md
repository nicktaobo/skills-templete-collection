# 设计原则速查

## 1. 设计不是好看，是信任感

用户不会先研究功能。他先判断这个页面是否专业、可信、值得继续看。

## 2. 信息层级

标题最大，副标题次之，CTA 最醒目。Feature 描述不能和标题一样抢眼。

## 3. 留白

留白让信息呼吸。拥挤不是信息丰富，是难读。

## 4. 对比度

小字也必须看得清。CTA 必须和背景有强对比。

## 5. 一致性

一致性不是所有东西完全一样，而是同类元素遵守同一套规则。

## 6. 移动端优先

大部分流量来自手机。手机端不是桌面端缩小版。

## 7. 反 AI 味

竞品和 AI 默认做的，你换一种。

- 不用默认字体
- 不用紫蓝渐变白背景
- 不用居中 Hero + 3 列 Features
- 不用 emoji icon
- 不用抽象 3D 球体
- 不写空泛标题
